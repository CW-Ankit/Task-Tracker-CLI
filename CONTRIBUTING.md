# Contributing to Task Tracker CLI 🤝

First off, thank you for considering contributing to this project! It's always appreciated.

## 🛠️ How to Contribute

### 1. Setup Your Environment
- Clone the repository.
- Run `npm install` (if applicable).
- Run `npm link` to install the CLI globally on your machine for testing.

### 2. Making Changes
- **Create a Branch**: Always create a new branch for your feature or bug fix: `git checkout -b feature/my-new-feature`.
- **Write Tests**: If you add a new feature, please add a corresponding test in the `tests/` folder.
- **Style Guide**: We use Prettier and ESLint. Please ensure your code is formatted correctly before submitting.

### 3. Submitting Your Changes
- **Commit Messages**: Use clear, descriptive commit messages (e.g., `feat: add priority levels to tasks`).
- **Pull Request**: Open a Pull Request (PR) with a detailed description of what you changed and why.

## 📜 Code of Conduct
Please be respectful and kind in all interactions. This project is a learning resource, and we welcome developers of all skill levels.

Happy coding! 🚀
